Dapatkan di console.anthropic.com → API Keys. Hanya tersimpan di browser ini.

Ikuti panduan di tab Apps Script untuk mendapatkan URL ini.

Nama tab pada Google Spreadsheet tujuan. Akan dibuat otomatis jika belum ada.

💡 Semua konfigurasi tersimpan di browser masing-masing pengguna. Setiap teman yang menggunakan scanner ini perlu mengisi API Key dan URL mereka sendiri di halaman yang sama.
🚀 Ikuti langkah ini sekali saja. Setelah selesai, scanner bisa langsung dipakai untuk mengirim data ke Spreadsheet kamu.
  1. Buka Google Spreadsheet yang ingin kamu gunakan, lalu klik menu Ekstensi → Apps Script
  2. Hapus semua kode yang ada, kemudian tempel (paste) seluruh kode di kotak bawah ini
  3. Klik tombol Simpan (ikon disket) lalu klik Deploy → New deployment
  4. Pilih type: Web app → Execute as: Me → Who has access: Anyone → klik Deploy
  5. Izinkan akses ketika diminta, lalu salin URL yang muncul dan tempel di tab Konfigurasi
